## changelog

### V1.0

前台页面表格展现和基本操作的 动态grid 

. 支持 静态、动态 -- 数据加载(包括jsonp)
. 支持 静态、动态 -- 数据 本地翻页、排序
. 支持 自定义列 样式 和 结构
. 丰富的 表格对象、数据对象 事件 管理等

后续升级：
. 支持表格自定义tr模板，以解决 复杂个性化 渲染问题
